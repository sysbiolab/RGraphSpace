# Toy 'igraph' objects

Small 'igraph' objects used for workflow demonstrations. All graphs
include 'x', 'y', and 'name' vertex attributes.

## Usage

``` r
data(gtoy1)
```

## Format

igraph

## Source

This package.

## Value

A pre-processed igraph object.

## Examples

``` r
library(RGraphSpace)
data(gtoy1)
data(gtoy2)
```
